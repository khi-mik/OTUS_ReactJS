## Lesson 2 - hometask:
Console calculator with basic arithmetic operations (+ - / *)
- write an expression after > symbol 
- use spaces in an expressions
> 2 + 2 * 2
Result: 6

- maintain math operators 
** - squaring (2 ** 2 = 4)
! - factorial (4! = 1 * 2 * 3 * 4)
sin(x)
cos(x)
tan(x)
fib(number) - fibonacchi



install dependencies
```npm install``` 

run calculator
```npm run calc```

run tests
```npm run test```

run linter to check code
```npm run lint```

run linter to fix code
```npm run lint-fix```



Examples:

```
> 10 + 10
Result: 20
> 10 + 10 * 20 - 30
Result: 180
> 19 + -10
Result: 9
> 2 ** + 1
Result: 5
> cos 0
Result: 1
```
